# 化妆品备案安全评估工具

一款专为化妆品企业设计的**一站式智能安全评估工具**，帮助您高效管理原料库、设计配方、进行合规性检查，并生成符合监管要求的安全评估报告。

## 📋 功能特点

### 核心功能模块

| 模块 | 功能说明 |
|------|----------|
| **配方设计** | 可视化配方编辑，实时计算成分含量，支持复合原料拆分 |
| **原料库管理** | 支持批量导入、自动匹配INCI名称、供应商信息管理 |
| **目录同步** | 自动抓取国家药监局已使用化妆品原料目录（目录I/II） |
| **合规性检查** | 自动校验原料是否符合《化妆品安全技术规范》 |
| **风险物质识别** | 自动识别配方中可能含有的风险物质（二噁烷、二甘醇等） |
| **安全评估引擎** | 基于TTC方法、NOAEL/MOE法进行智能安全评估 |
| **报告生成** | 一键生成符合备案要求的安全评估报告（含三大表） |

### 安全评估特性

- ✅ **多层合规检查**：技术规范强制检查 → 已使用原料核查 → 国际权威评估结论校验
- ✅ **智能风险识别**：基于规则引擎自动识别风险物质
- ✅ **多人群暴露评估**：支持婴幼儿、儿童、成人、孕妇等多人群分层评估
- ✅ **权威数据优先**：优先引用CIR、SCCS等权威机构评估结论
- ✅ **报告自动生成**：支持完整版/基本结论报告，符合《技术导则》要求

## 📖 功能模块说明

### 一、配方设计

配方设计模块用于创建和编辑化妆品配方，支持复合原料拆分和多成分展开显示。

#### 配方表格列说明

| 列名 | 说明 |
|------|------|
| 原料序号 | 原料在配方中的顺序编号 |
| 标准中文名称 | 原料的中文名称（INCI标准中文名） |
| INCI名称 | 国际化妆品原料命名 |
| 添加量(%) | 该原料在配方中的添加百分比 |
| 原料中成分含量(%) | 复合原料中各成分的占比 |
| 实际成分含量(%) | 计算后的实际浓度 |
| 主要使用目的 | 原料的使用目的 |
| 原料报送码 | 原料报送时的唯一标识码 |
| 禁用/限用 | 合规性检查结果 |
| SED | 全身暴露量计算值 |
| 综合结果 | 最终评估结论 |

#### 复合原料处理

复合原料会自动拆分为多行显示，主行展示原料基本信息，子行展示各成分详细信息。合并列（原料名称、添加量、禁用、限用等）会自动跨多行合并，提升可读性。

### 二、原料库管理

管理企业的原料数据库，支持搜索、添加、编辑、删除和批量操作。

#### 导入模板格式

| 列名 | 说明 | 必填 |
|------|------|------|
| 原料名称 | 原料的中文名称 | ✅ |
| 成分组成 | 复合原料的成分构成（如：水(50%)、甘油(50%)） | ❌ |
| 内部代码 | 企业内部编码 | ❌ |
| 主要使用目的 | 原料的使用目的 | ❌ |
| 原料报送码 | 原料报送码 | ❌ |

### 三、合规性检查

自动执行多层合规检查：

1. **第一层**：《化妆品安全技术规范》强制检查（禁用/限用/准用组分）
2. **第二层**：已使用原料及上市使用量核查
3. **第三层**：国际权威评估结论校验（CIR/SCCS）
4. **第四层**：其他证据类型判定

### 四、风险物质识别

基于规则引擎自动识别风险物质：

| 触发原料 | 风险物质 |
|----------|----------|
| 卡波姆 | 苯残留 |
| 聚丙烯酰胺 | 丙烯酰胺单体 |
| PEG类 | 二噁烷、二甘醇 |
| 植物提取物 | 农药残留、重金属 |

### 五、安全评估引擎

**评估方法优先级**：
1. 《技术规范》限用/准用组分（必须优先满足）
2. 国际权威机构评估结论（CIR/SCCS）
3. 历史使用量数据
4. NOAEL/MOE法
5. TTC方法（兜底）

**TTC分类**：
- Class I（低关注）：1800 μg/kg bw/day
- Class II（中等关注）：540 μg/kg bw/day
- Class III（高关注）：90 μg/kg bw/day

### 六、报告生成

支持多种报告模板：

| 模板 | 适用范围 |
|------|----------|
| CN-FULL | 中国完整版安全评估报告 |
| CN-SUMMARY | 中国安全评估基本结论 |
| CN-CHILDREN | 中国完整版报告（儿童专用） |
| EU-CPSR | 欧盟CPSR报告 |

## ⚠️ 注意事项

1. **首次同步**：首次使用建议先同步目录I和目录II，以便INCI匹配功能正常工作
2. **网络要求**：同步目录功能需要网络连接
3. **数据备份**：定期备份 `userdata` 目录以防止数据丢失
4. **Edge浏览器**：同步功能依赖Edge浏览器，请确保已安装
5. **报告审核**：生成的评估报告仅供参考，最终需经专业人员审核
