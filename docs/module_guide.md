## 5. 安全评估引擎

**文件**: `safety_engine.py`

### 功能说明

安全评估核心逻辑：
- 多层合规性检查
- 风险物质识别
- 暴露评估计算
- TTC/MOE评估方法

### 评估流程

```
输入: 配方数据 + 产品参数
    │
    ├─→ 第一层: 《技术规范》强制检查
    │      └─→ 禁用组分检查 → 限用组分检查 → 准用组分检查
    │
    ├─→ 第二层: 已使用原料核查
    │      └─→ 目录匹配 → 历史使用量比对
    │
    ├─→ 第三层: 国际权威评估结论校验
    │      └─→ CIR/SCCS结论匹配 → 限制条件解析
    │
    └─→ 第四层: 安全评估计算
           └─→ SED计算 → MOE/TTC评估 → 综合结论
```

### 核心函数

| 函数 | 说明 |
|------|------|
| `assess_formula()` | 评估整个配方 |
| `assess_ingredient()` | 评估单个原料 |
| `calculate_sed()` | 计算全身暴露量 |
| `check_compliance()` | 合规性检查 |
| `identify_risk_substances()` | 风险物质识别 |

### 评估方法优先级

1. **《技术规范》限用/准用组分**（必须优先满足）
2. **国际权威机构评估结论**（CIR/SCCS）
3. **历史使用量数据**
4. **NOAEL/MOE法**
5. **TTC方法**（兜底）

| 模板ID | 报告类型 | 适用范围 |

| CN-CHILDREN | 儿童专用完整版报告 | 儿童化妆品 |

## 7. 产品分类判定模块

**文件**: `product_classifier.py`

### 功能说明

根据产品参数判定产品类别：
- 第一类/第二类判定
- 儿童化妆品识别
- 特殊产品类型识别（如纳米原料、新原料）

### 判定逻辑

```
输入: 使用部位、使用方法、适用人群、功效宣称
    │
    ├─→ 特殊化妆品判定
    │   └─→ 祛斑美白/防晒/染发/烫发/防脱发 → 第一类
    │
    ├─→ 特殊人群/原料判定
    │   ├─→ 含婴幼儿/儿童 → 第一类
    │   └─→ 含监测期新原料 → 第一类
    │
    └─→ 第二类情形判定
        ├─→ 含纳米原料 → 第二类情形一
        ├─→ 非防晒类用表5未收载防晒剂 → 第二类情形一
        ├─→ 功效为祛痘/抗皱/除臭/去屑/脱毛/去角质 → 第二类情形一
        ├─→ 剂型为贴膜/气雾剂 → 第二类情形一
        └─→ 其他 → 第二类情形二
```

### 核心函数

| 函数 | 说明 |
|------|------|
| `classify_product()` | 产品分类判定 |
| `is_children_product()` | 判断是否儿童产品 |
| `get_required_endpoints()` | 获取必需的毒性终点 |
